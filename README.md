# Surface Theory Landing Page

Professional landing page for Surface Theory - Premium Interior Surfaces company.

## Features

- Modern design with premium surface imagery
- Lead capture form with MEGA integration
- Mobile responsive
- Conversion-optimized layout
- Phone number tracking
- Form validation

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- MEGA Component Library

## Deployment

This project is set up for deployment on Vercel with the MEGA team account.

Repository: https://github.com/Landing-Pages-Websites/surface-theory-landing

## Development

```bash
npm install
npm run dev
```
